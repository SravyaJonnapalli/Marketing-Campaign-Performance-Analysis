# Marketing-Campaign-Performance-Analysis
**Project Overview**
This project focuses on analyzing marketing campaign data to evaluate campaign performance, customer engagement, ROI trends, and cost efficiency using Python, SQL, and Power BI.

The project includes:
- Data cleaning and preprocessing using Python
- SQL-based analytical queries
- Interactive Power BI dashboard
- Business insights and trend analysis

**Tools & Technologies**
- Python
- MySQL
- Power BI

**Dataset Features**
The dataset contains:
- Campaign information
- Campaign type and channel
- Customer segments and demographics
- Clicks and impressions
- ROI and conversion rates
- Acquisition cost and profit
- Engagement metrics
- Time-based campaign data

**Data Preprocessing (Python)**
Performed:
- Date format conversion
- Click through rate and CPC calculation
- Data type correction
- Cost data cleaning
- Time-based column extraction

Created additional fields:
- Gender
- Age_Group
- Duration_Days
- Click Through Rate
- Cost per Click
- Cost per Impression
- Year, Month, Month_Name

** SQL Analysis**
Performed analysis on:
- Campaign-wise ROI
- Location-wise performance
- Customer segment analysis
- Monthly ROI trends
- Engagement and CTR analysis
- Cost efficiency analysis
- Top-performing campaigns

Used:
- GROUP BY
- Aggregate functions
- Window functions
- Ranking functions
- Filtering and sorting

**Power BI Dashboard**
Created a 3-page interactive dashboard:

**Overview**
- KPI Cards
- ROI Analysis
- Customer Segment Distribution
- Location Performance

**Performance Analysis**
- Monthly ROI Trends
- Cost vs ROI Scatter Analysis
- CTR & Engagement Analysis

**Campaign Insights**
- Top Campaigns
- Location-wise ROI
- Interactive Filters & Slicers

**Key Insights**
- ROI remained relatively consistent across campaign types
- Customer segments showed varying engagement patterns
- CTR and engagement metrics provided deeper campaign insights
- Certain channels performed better in specific locations
- Cost efficiency varied across campaign categories

**Project Outcome**
This project demonstrates end-to-end data analytics workflow:
- Data cleaning
- SQL analysis
- Business intelligence dashboarding
- Insight generation
